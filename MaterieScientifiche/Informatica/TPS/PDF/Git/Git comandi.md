### Comandi Git principali (locale, remoto, branch, fork, inizializzazione)

| **Azione**                                                       | **Comando**                                                                                                                                            | **Note**                                                                 |
| ---------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------ |
| **ðŸ”¹ Inizializzare Git (solo la prima volta)**                   | `git config --global user.name "TuoNome"``git config --global user.email "tuo@email.com"`                                                              | Configura Git con il tuo nome e email (necessario per i commit).         |
| **ðŸ”¹ Creare un nuovo repository Git locale**                     | `git init`                                                                                                                                             | Crea un repository Git nella cartella corrente.                          |
| **ðŸ”¹ Aggiungere file al repository**                             | `git add nomefile``git add .`                                                                                                                          | Aggiunge i file al prossimo commit.                                      |
| **ðŸ”¹ Salvare le modifiche con un commit**                        | `git commit -m "Messaggio del commit"`                                                                                                                 | Crea un nuovo commit con il messaggio fornito.                           |
| **ðŸ”¹ Visualizzare lo stato del repository**                      | `git status`                                                                                                                                           | Mostra file modificati e in attesa di commit.                            |
| **ðŸ”¹ Visualizzare la cronologia dei commit**                     | `git log``git log --oneline``git log --graph --all`                                                                                                    | Mostra la storia dei commit.                                             |
| **ðŸ”¹ Annullare modifiche (rollback)**                            | `git checkout -- nomefile``git reset HEAD nomefile``git reset --hard HEAD~1`                                                                           | Ripristina file o annulla l'ultimo commit.                               |
| **ðŸ”¹ Creare un nuovo branch**                                    | `git branch nomeBranch`                                                                                                                                | Crea un nuovo branch senza spostarsi su di esso.                         |
| **ðŸ”¹ Passare a un branch esistente**                             | `git checkout nomeBranch`                                                                                                                              | Cambia il branch attivo.                                                 |
| **ðŸ”¹ Creare e passare a un nuovo branch**                        | `git checkout -b nomeBranch`                                                                                                                           | Crea e passa al nuovo branch.                                            |
| **ðŸ”¹ Vedere tutti i branch**                                     | `git branch``git branch -a`                                                                                                                            | `-a` mostra anche i branch remoti.                                       |
| **ðŸ”¹ Eliminare un branch**                                       | `git branch -d nomeBranch``git branch -D nomeBranch`                                                                                                   | `-D` forza l'eliminazione.                                               |
| **ðŸ”¹ Unire un branch al branch attuale**                         | `git merge nomeBranch`                                                                                                                                 | Unisce `nomeBranch` al branch attuale.                                   |
| **ðŸ”¹ Clonare un repository remoto**                              | `git clone url-repo`                                                                                                                                   | Scarica una copia del repo remoto sul PC.                                |
| **ðŸ”¹ Aggiungere un repository remoto**                           | `git remote add origin url-repo`                                                                                                                       | Collega il repository locale a un repository remoto.                     |
| **ðŸ”¹ Visualizzare repository remoti**                            | `git remote -v`                                                                                                                                        | Mostra i remoti configurati.                                             |
| **ðŸ”¹ Scaricare modifiche dal repository remoto**                 | `git fetch``git pull`                                                                                                                                  | `fetch` scarica senza applicare, `pull` scarica e applica.               |
| **ðŸ”¹ Inviare modifiche al repository remoto**                    | `git push origin nomeBranch`                                                                                                                           | Carica il branch sul repository remoto.                                  |
| **ðŸ”¹ Eliminare un branch remoto**                                | `git push origin --delete nomeBranch`                                                                                                                  | Rimuove un branch dal repository remoto.                                 |
| **ðŸ”¹ Creare un fork di un repository (GitHub/GitLab)**           | ðŸ”¸ **1.** Vai su GitHub/GitLab e clicca "Fork".                                                                                                        | Crea una copia del repository originale nel tuo account.                 |
| **ðŸ”¹ Clonare un repository forkato**                             | `git clone url-tuo-repo-forkato`                                                                                                                       | Scarica il fork sul tuo PC.                                              |
| **ðŸ”¹ Aggiungere il repository originale come upstream**          | `git remote add upstream url-repo-originale`                                                                                                           | Collega il fork al repository originale.                                 |
| **ðŸ”¹ Verificare i remoti (origin e upstream)**                   | `git remote -v`                                                                                                                                        | Mostra `origin` (fork) e `upstream` (repo originale).                    |
| **ðŸ”¹ Sincronizzare il fork con lâ€™originale**                     | `git fetch upstream``git checkout main``git merge upstream/main``git push origin main`                                                                 | Scarica e unisce gli aggiornamenti dal repository originale al tuo fork. |
| **ðŸ”¹ Creare una Merge Request (GitLab) o Pull Request (GitHub)** | ðŸ”¸ **1.** Pusha le modifiche sul tuo fork: `git push origin nomeBranch`ðŸ”¸ **2.** Vai su GitHub/GitLab e apri una **Pull Request** o **Merge Request**. | Permette di proporre modifiche al repository originale.                  |
